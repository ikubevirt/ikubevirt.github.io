
## 文件存储

**典型设备** FTP、NFS 服务器，为了克服块存储文件无法共享的问题，所以有了文件存储，在服务器上架设 FTP 与 NFS 服务器，就是文件存储。

**优点**

造价低，随便一台机器就可以了
方便文件可以共享

**缺点**

- 读写速率低
- 传输速率慢

**使用场景**

- 日志存储
- 有目录结构的文件存储
- ...

## 对象存储

**典型设备**

内置大容量硬盘的分布式服务器(swift, s3)；多台服务器内置大容量硬盘，安装上对象存储管理软件，对外提供读写访问功能。

**优点**

- 具备块存储的读写高速。
- 具备文件存储的共享等特性

**使用场景**：(适合更新变动较少的数据)

- 图片存储
- 视频存储
- ...
